# FolderSync
一个轻量级の基于MD5文件查重の文件管理系统，实现文件的备份，文件历史版本的还原，etc<br/>
还有一规划加上百度云盘的文件链接的，但是坑太大怕填不上……<br/>
<br/>
因为感觉加上个SQL保存文件数据可以提升一下文件索引的性能，所以目前代码全部打回地狱重写中<br/>
